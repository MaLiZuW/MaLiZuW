🧠 CephaloLog – Inštalácia na Android

Ako nainštalovať aplikáciu na telefón cez GitHub Pages (zadarmo)

Čo budeš potrebovať

  • Účet na GitHub (zadarmo – github.com)
  • Súbory z tohto balíka: headache-tracker.html, manifest.json, sw.js, icon-192.png, icon-512.png
  • Chrome na Androide

Postup krok za krokom

1

Vytvor nový GitHub repozitár

Prihlás sa na github.com → klikni + → New repository. Názov napr. cephalolog, nastav na Public, klikni Create repository.

2

Nahraj všetky súbory

V novom repozitári klikni uploading an existing file a nahraj všetkých 5 súborov naraz: headache-tracker.html, manifest.json, sw.js, icon-192.png, icon-512.png. Klikni Commit changes.

3

Zapni GitHub Pages

Choď do Settings → Pages (v ľavom menu). Pod Source vyber Deploy from a branch, branch main, priečinok / (root). Klikni Save.

4

Počkaj minútku a skopíruj link

Po chvíli sa zobrazí zelený banner s URL v tvare:
https://tvoje-meno.github.io/cephalolog/headache-tracker.html
Skopíruj si ho.

5

Otvor link v Chrome na Androide

Na telefóne otvor Chrome a zadaj skopírovanú URL. Nechaj stránku načítať.

6

Nainštaluj aplikáciu

Chrome automaticky zobrazí banner „Pridať CephaloLog na plochu" – klikni naň. Alebo: menu Chrome (tri bodky) → Pridať na plochu → Inštalovať.

✅ Hotovo! Na ploche sa objaví ikona CephaloLog. Aplikácia sa otvára na celú obrazovku bez lišty prehliadača, funguje offline a dáta sú uložené lokálne v telefóne.
⚠️ Dôležité: Dáta sú uložené v localStorage prehliadača. Pravidelne exportuj zálohu cez Nastavenia → Exportovať dáta (JSON) a ulož si ju.

Alternatívne možnosti hostingu

Netlify Drop (najrýchlejšie)

Choď na app.netlify.com/drop a pretiahni priečinok so všetkými 5 súbormi. Za 30 sekúnd máš link. Nevyžaduje registráciu.

Vercel

Podobné ako Netlify. Registruj sa na vercel.com, nahraj súbory a dostaneš link.
